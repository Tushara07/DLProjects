# Breast Cancer Prediction using ANN

An end-to-end Deep Learning project using Artificial Neural Networks (ANN) and Streamlit.

## Features

- Breast Cancer Wisconsin Dataset
- ANN with TensorFlow/Keras
- StandardScaler for preprocessing
- Streamlit Web Application
- Binary Classification:
    - Benign
    - Malignant

## Tech Stack

- Python
- TensorFlow / Keras
- Scikit-learn
- Pandas
- Streamlit

## Run Locally

pip install -r requirements.txt

streamlit run app/app.py
